# TEOL AI Asistan

Modern AI chatbot with TEOL corporate design, built with Flask and deployed on Vercel.

## Features
- 🎨 TEOL Corporate Design
- 🤖 Gemini AI Integration  
- 🎙️ Voice Recognition
- 🔊 Text-to-Speech
- 📱 Responsive Design
- ⚡ Serverless Architecture

## Deployment
Deployed on Vercel with Python runtime.

## Environment Variables
```
GEMINI_API_KEY=your_gemini_api_key_here
```

## Local Development
```bash
pip install -r requirements.txt
python app.py
```

Visit: http://localhost:5000

## Tech Stack
- Backend: Python Flask
- Frontend: TailwindCSS + Vanilla JS
- AI: Google Gemini
- Deployment: Vercel
- Icons: Lucide

## API Endpoints
- `GET /` - Ana sayfa
- `POST /api/chat` - Sohbet API'si
- `GET /api/status` - Durum kontrolü
